# Data Source

**OCHA Financial Tracking Service (FTS)**  
Dataset: `ukr-requirements-and-funding-data`  
[Download from HDX](https://data.humdata.org/dataset/ukr-requirements-and-funding-data)

## Fields (which I used in queries)

- `year` — Calendar year
- `cluster` — Humanitarian sector name
- `requirements_usd` — Planned funding
- `funding_usd` — Actual funding received
